Trabalhando com listas

Listas ordenatas A ordem fas a diferença.

Para deixar uma lista de forma Ordenada se ultiliza a TAG <ol>.

Na lista ordenada pode se usar 5 tipos de ordenação.
1 type "1".
2 type "A".
3 type "a".
4 type "I".
5 type "i".

Por padrão o type é o "A", para alterar tem que por o seguinte parametro. <ol type="{uma das cinco opções}">.

Para deixar a lista ordenada de forma númerica, usar parametro type da seguinte forma. <ol type="1">.

  1. Acordar
  2. Ligar para o João
  3. Tomar café

Para deixar uma lista ordenada de forma alfabétida Maiúsculas, usar o parametro type da seguinte forma. <ol type="A">.

  1. Acordar
  2. Ligar para o João
  3. Tomar café

Para deixar uma lista ordenada de forma alfabética minúsculas, usar o parametro type da senguinte forma. <ol type="a">.

  1. Acordar
  2. Ligar para o João
  3. Tomar café

Para deixar uma lista ordenada com algarismo romanos em Maiúsculas, usar o parametro type da seguinte forma. <ol type="I">.

  1. Acordar
  2. Ligar para o João
  3. Toma café

Para deixar uma lista ordenada com algarismo romano em minúsculas <ol type="i">.

  1. Acordar
  2. Ligar para o João
  3. Tomar café

Listas NÃO ordenatas

Essas lista é usada com a ondem da lista não faz sentido/não precisa enúmerar.

Na lista não ordenada pode ser usada 3 tipos de marcações.
Por padrão o type é "disc", para alterar tem que por o seguinte parametro. <ul type="{uma das três opções}">.

1 type "disc".
2 type "circle".
3 type "square".

  • pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Para alterar o type da lista, usar o parametro TYPE da seguinte forma. <ul type="circle">.

  • pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Para alterar o type da lista, usar o parametro TYPE da seguinte forma. <ul type="square">.

  • pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
